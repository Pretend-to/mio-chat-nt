## 简介
仿 qqnt 的 gpt webUI，由 [@lss233](https://github.com/lss233) 的优秀开源项目 [chatgpt-mirai-qq-bot](https://github.com/lss233/chatgpt-mirai-qq-bot) 提供后端支持。

## features
- markdown 支持
- 对话
- 音频/图像支持
- 移动端支持 (优化中) 
- 访问/管理码
- 聊天记录以图像分享
